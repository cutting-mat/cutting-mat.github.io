# 快速上手

## 初始化项目

### npm

使用npm初始化CuttingMat项目。

``` bash
npm init @cutting-mat
```

### git

从 git repository 拉取项目模板。

``` bash
git clone https://github.com/cutting-mat/template-element-ui.git
```

## 启动项目

安装依赖并运行开发服务。

``` bash
npm i
npm run serve
npm run build
```

## 字体图标

CuttingMat 使用 [iconfont](https://www.iconfont.cn/) 字体图标库。
